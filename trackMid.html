    Access profiles of military personnels

    This window allows you to access the profile of our troops stationed in various parts of the world. Each military officer can be identified with a DoD Identification Number (a unique number, example: 1401700000), the DoD Identification Number can be used to query profile details of an officer (...we can only provide brief info about a soldier for security reasons best known to us). The Bundeswehr Military Command has established this security measure to cub crimes committed in the name of the Bundeswehr Military officers in general. Please be warned, ensure to verify a soldier's profile before identifying such individual as a real Bundeswehr.
